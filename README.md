# project-vue
Проект на Vue.js содержит реализацию следующих возможностей:
- хранилище
- роутер
- ленивая маршрутизация
- фоновые потоки webWorkers с исполнением запросов на "бэк" и реализацией "тяжелых" вычислений
- верстка flexbox
- css3 
- анимация
- таблица с бесконечным scroll
- потоковое радио
- форма для оплаты

## Project setup
```
yarn install
```
